Eventos DOM

DOM é a sigla pra: Document Object Model, e são objetos JS que servem para manipular os elementos dos documentos HTML ou CSS.

A Árvore DOM é básicamente uma árvore hierarquica dos objetos DOM que serve para especificar elementos.

  • getElementsByTagName
  • getElementsByName
  • getElementByID
  • getElementsByClassName
  • document.querySelector
  • document.querySelectorAll
